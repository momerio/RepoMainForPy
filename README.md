A sample program for repository mining using GitPython.

## development environment
- Python 3.10.0
- git version 2.29.2.windows.3
- GitPython  3.1.24

## pip install
```
$ pip install GitPython
```

- [GitPython Repository](https://github.com/gitpython-developers/GitPython)
